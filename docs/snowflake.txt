2. Grant Minimal Privileges
Grant only the necessary privileges for creating and managing databases and schemas:

       -- Allow creation of databases
       GRANT CREATE DATABASE ON ACCOUNT TO ROLE TERRAFORM_MANAGER;

       -- Allow creation of schemas in all databases (or specify a database)
       GRANT CREATE SCHEMA ON ALL DATABASES IN ACCOUNT TO ROLE TERRAFORM_MANAGER;

       -- Allow usage on all databases (needed to access databases)
       GRANT USAGE ON ALL DATABASES IN ACCOUNT TO ROLE TERRAFORM_MANAGER;

       -- Optionally, grant ownership on schemas they create
       GRANT ALL ON ALL SCHEMAS IN DATABASE <database_name> TO ROLE TERRAFORM_MANAGER;
       
       
Note: Replace <database_name> with the actual database name if you want to restrict to a specific database.

3. Assign the Role to the User

       -- Create the user (if not already exists)
       CREATE USER TERRAFORM_USER PASSWORD = '...';

       -- Assign the role to the user
       GRANT ROLE TERRAFORM_MANAGER TO USER TERRAFORM_USER;

       -- Set the role as default for the user
       ALTER USER TERRAFORM_USER SET DEFAULT_ROLE = TERRAFORM_MANAGER;
